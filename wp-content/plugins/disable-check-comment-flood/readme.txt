== Description ==

Disables the Check Comment Flood feature so comments can be post-dated. If your visitors are receiving the message, "You are posting comments too quickly. Slow down." while posting, this plugin disables that function.

== Installation ==

1. Upload `disable-check-comment-flood.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= Is it a good idea to disable the Check Comment Flood feature? =

There are pros and cons, but you may find it useful to disable the feature if you edit comments or have a different way of using comments.
